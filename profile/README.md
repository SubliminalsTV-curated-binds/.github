# 🕹️ SubliminalsTV — Curated Joystick Bindings

The open-source home for my **Star Citizen** joystick binding configs. Each stick I support lives in its own repository — clone only the one you fly.

> 💎 **Want stable, tested, plug-and-play binds?** Get them from **[subliminal.gg/bindings](https://subliminal.gg/bindings/)**.
>
> This org is the **bleeding-edge / contributor** side — files here can be experimental or mid-patch.
