# 🕹️ SubliminalsTV — Curated Joystick Bindings

The open-source home for my **Star Citizen** joystick binding configs. Each stick I support lives in **its own repository** — so you clone only the one you fly.

> 💎 **Want stable, tested, plug-and-play binds?** Get them from **[subliminal.gg/bindings](https://subliminal.gg/bindings/)**.
>
> This org is the **bleeding-edge / contributor** side. Files here can be experimental or mid-patch — built for advanced users comfortable with Joystick Gremlin and Star Citizen control profiles.

## Why one repo per stick?

Almost nobody owns every stick I cover — so you shouldn't have to download every stick's files to help with yours. Each repo is **self-contained** (the Joystick Gremlin R14 profile, the Star Citizen layout XML, the Bindings Toolkit, and audio files), so you clone just your stick and contribute fixes back with a focused pull request.

## Contributing

On each major Star Citizen patch I clear baseline binds, validate, and coordinate testing priorities on **[Discord](https://subliminal.gg)**. PRs against your stick's repo are welcome — I verify, test, and package the stable releases for the website.
